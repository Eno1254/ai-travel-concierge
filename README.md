# 🩺 AI Medical Assistant

An AI-powered medical assistant that can answer general health questions and analyze uploaded medical reports using Retrieval-Augmented Generation (RAG).

⚠️ This project is for educational purposes only and does not replace professional medical advice.

---

## Features

- 💬 Chat interface for medical questions
- 📄 Upload medical reports (PDF)
- 🧠 AI reads and explains report contents
- 🔍 Uses RAG 
- 📊 Vector database using FAISS
- ⚡ Powered by Groq LLM
- 🌐 Deployed on Streamlit Cloud
